* Convert object to table html

This library is no longer under maintenance. Please use [[http://github.com/akicho8/html_format][html_format]] instead.

** Examples

#+BEGIN_SRC ruby
quick_table { {:a => 1, :b => 2} }
# => [a][1]
# => [b][2]

quick_table { [{:a => 1, :b => 2}, {:a => 3, :b => 4}] }
# => [a][b]
# => [1][2]
# => [3][4]

quick_table { [[:a, :b], [ 1,  2], [ 3,  4]] }
# => [a][b]
# => [1][2]
# => [3][4]

quick_table { [:a, :b] }
# => [a][b]

quick_table { :a }
# => [a]
#+END_SRC
